### Emails and SMS from command line
E-mail:
```bash
echo 'Hi this is a message' | mail -s 'subject' 'email_address'
```
SMS:
```bash
echo 'Sent from my terminal!' | mail -s 'SMS sent from linux' '81812345678@vtext.com'
```
